#Music Player project.


A music streaming application (simillar to a Spotify) built with Node.js, Express, and MySQL for database.

#Features


-Player with skip, shuffle and repeat playlist buttons (styling for buttons and animation features were taken from https://codepen.io/havardob/pen/MWqMmgb)

-User registration and login with bcrypt password hashing

-Song search

-Favorites library

-Admin panel with managing, uploading and deleting tracks

-Live radio stations stream (was done thanks to 90k radio stations API https://radio.herani.id/)

-Light/dark theme toggle (was taken from https://codepen.io/havardob/pen/MWqMmgb as well)

It is a Web Programming project for University of Messina
